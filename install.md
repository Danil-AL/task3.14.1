# Установка Git

# Windows
1. Перейдите на [официальный сайт Git](https://git-scm.com/download/win).
2. Скачайте установщик.
3. Запустите файл → следуйте мастеру установки (настройки по умолчанию).

# macOS
1. Откройте Terminal.
2. Установите через Homebrew:  
   ```bash
   brew install git 

# Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install git
