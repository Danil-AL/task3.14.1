## Основные команды Git

### Таблица команд

| Команда | Описание | Пример |
|--------|---------|-------|
| `git status` | Показать состояние файлов (изменены, новые, удалённые) | `git status` |
| `git add .` | Добавить все изменённые файлы в индекс | `git add .` |
| `git add file.txt` | Добавить конкретный файл | `git add README.md` |
| `git commit -m "текст"` | Сохранить изменения с комментарием | `git commit -m "add: инструкция по установке"` |
| `git push` | Отправить изменения на GitHub | `git push origin main` |
| `git pull` | Получить обновления с GitHub | `git pull origin main` |
| `git log` | Посмотреть историю коммитов | `git log --oneline` |
