## Создание репозитория


### Шаг 1. На GitHub


1. Зайдите на [github.com](https://github.com) и авторизуйтесь.  
2. Нажмите «+» → «New repository».  
3. Заполните поля:  
   - **Repository name**: `task3.14.1`  
   - **Description**: «Мой первый Git-проект»  
   - Поставьте галочку **«Add a README file»**  
4. Нажмите «Create repository».

### Шаг 2. Локально (на компьютере)

1. Создайте папку: `task3.14.1` (например, в `C:\projects\` или `~/projects/`).  
2. Откройте терминал в этой папке.  
3. Инициализируйте Git:
