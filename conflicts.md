## Разрешение конфликтов в Git


Конфликты возникают при попытке слить ветки, если Git не может автоматически объединить изменения в одном и том же файле.



### Шаги решения

1. **Откройте файл** в редакторе (например, в VS Code).  
2. **Удалите маркеры** конфликта (`<<<<<<<`, `======`, `>>>>>>>`).  
3. **Выберите нужный вариант** кода или объедините изменения вручную.  
4. **Сохраните файл**.  
5. **Добавьте изменения в индекс**:  
   ```bash
   git add путь/к/файлу.txt
